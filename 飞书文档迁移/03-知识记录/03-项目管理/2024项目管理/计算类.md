成本类
 基础公式
PV = 计划干的活 = 计划完成进度 = 计划完成的工作量或价值量 × 计划单价  = 预算 * 到目前为止计划完成的工作比例。
EV = 实际干完的活 =挣值 = 实际完成的工作量或价值量 × 实际单价 = 预算 * 实际完成的工作比例
AC = 实际花费  =  实际完成的工作量或价值量 × 计划单价 = 实际发生的全部成本之和
BAC = 总预算
EAC=完工估算 = AC + ETC  =  AC+（ BAC - EV ） / CPI= BAC/CPI，
ETC=完工尚需估算 = （ BAC - EV ） / CPI
VAC 完工偏差 = BAC - EAC
管理公式
大于1，0都是好的，小于1,0都是不好的
CV = EV - AC = 成本差异 项目成本是否控制在预算之内
SV = EV - PV = 进度差异 项目进度是否按计划完成
CPI =  EV/AV = 成本绩效指数 当CPI > 1时，表明成本节约；CPI < 1则意味着成本超支。
SPI = EV/PV = 进度绩效指数 SPI > 1表示进度超前；SPI < 1则表示进度滞后。
典型是不纠偏的，非典型是纠偏的
